# Salle, tables & réservations

L'écran **Salle** vous permet de visualiser l'état de votre salle, d'ouvrir des commandes sur les tables et de gérer les réservations.

## Le plan de salle

Chaque table s'affiche avec une couleur indiquant son état :

- **Libre** (vert) — la table est disponible et peut accueillir des clients.
- **Occupée** (orange) — une commande est en cours sur cette table.
- **Addition demandée** (rouge) — le client a demandé l'addition ; la table attend l'encaissement.
- **Réservée** (bleu) — la table est bloquée pour une réservation à venir.

Cliquez sur une table pour voir le détail de la commande ou effectuer une action.

## Ouvrir une commande sur une table

- Cliquez sur une table **libre** pour l'ouvrir.
- Saisissez le nombre de couverts si demandé.
- Ajoutez des plats directement depuis le plan de salle : le menu s'affiche en panneau latéral.
- La commande est envoyée en cuisine dès qu'un article est ajouté.
- Vous pouvez ajouter des articles à une table **occupée** à tout moment.

## Demander l'addition et encaisser à la caisse

- Sur une table occupée, appuyez sur **Demander l'addition** pour passer la table en état « addition demandée ».
- L'encaissement peut se faire **immédiatement** depuis le plan de salle (bouton Encaisser) ou être **différé** en passant par l'écran Caisse.
- Une fois encaissée, la table repasse automatiquement à l'état **libre**.

## Réservations

- Appuyez sur **Nouvelle réservation** et renseignez : nom du client, date, heure, nombre de couverts et table souhaitée.
- Définissez la **durée** estimée de la réservation pour bloquer le créneau.
- Vous pouvez associer une **pré-commande** à la réservation pour préparer la cuisine à l'avance.
- Un **acompte** peut être enregistré lors de la création de la réservation.
- En cas d'annulation, le **remboursement** de l'acompte est saisi manuellement dans le module Dépenses.
