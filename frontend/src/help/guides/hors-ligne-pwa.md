# Mode hors-ligne & installation

Restoflow est une **Progressive Web App (PWA)** : elle peut être installée sur votre appareil et continuer à fonctionner même sans connexion Internet.

## Installer l'application

- Sur **Android** ou **Windows** : ouvrez Restoflow dans Chrome, puis appuyez sur **Installer l'application** dans la barre d'adresse ou dans le menu du navigateur.
- Sur **iOS (iPhone/iPad)** : ouvrez Restoflow dans Safari, appuyez sur le bouton **Partager**, puis sur **Ajouter à l'écran d'accueil**.
- Une fois installée, Restoflow s'ouvre comme une application native, sans barre de navigateur.
- L'installation n'est pas obligatoire : vous pouvez continuer à utiliser Restoflow dans votre navigateur habituel.

## Travailler hors-ligne (caisse)

- En cas de perte de connexion, un **bandeau orange** s'affiche en haut de l'écran pour vous avertir.
- L'écran **Caisse** reste entièrement fonctionnel hors-ligne : vous pouvez continuer à prendre des commandes et à encaisser.
- Les autres modules (tableau de bord, stock, paie…) nécessitent une connexion pour afficher des données à jour.
- Évitez de fermer l'application pendant une coupure réseau prolongée pour ne pas perdre les commandes en attente.

## Synchronisation à la reconnexion

- Dès que la connexion Internet est rétablie, Restoflow **synchronise automatiquement** toutes les commandes en attente.
- Un indicateur confirme la fin de la synchronisation : les données apparaissent dans le tableau de bord et le stock est mis à jour.
- En cas d'erreur de synchronisation (ex. conflit de données), une notification vous invite à vérifier les commandes concernées.

## Mises à jour

- Lorsqu'une nouvelle version de Restoflow est disponible, un **bandeau de mise à jour** s'affiche en haut de l'écran.
- Cliquez sur **Mettre à jour** pour recharger l'application avec la dernière version.
- Les mises à jour n'effacent pas vos données et ne nécessitent aucune installation manuelle.
- Si vous n'êtes pas certain que la mise à jour est sûre pendant le service, vous pouvez la reporter et l'appliquer entre deux services.
